# GramLite Flutter App

This is a Flutter app for a social media platform, built using the Dart programming language and the Flutter framework. The app provides users with the ability to post pictures and videos, follow other users, and engage with content through likes and comments. In addition to the todo list functionality.

## Features

- Authentication system with email and password.
- Create and edit user profiles.
- View user profiles and follow/unfollow users.
- Post pictures and videos.
- View a feed of posts from followed users.
- Like and comment on posts.

## Getting Started

To get started with this app, clone the repository to your local machine and open it in your preferred IDE or text editor.

```bash
git clone https://github.com/MOmoharrum/GramLite-Flutter-App.git
```

Before running the app, make sure to have Flutter and the necessary dependencies installed on your machine. Then, run the following command to install the required dependencies:

```bash
flutter pub get
```

Finally, start the app by running the following command:

```bash
flutter run
```
